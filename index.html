About Me

I'm a Geospatial Data Scientist with expertise in remote sensing, satellite imagery analysis, and machine learning applications for Earth observation. Currently working at SWAN as a GIS Analyst, I develop advanced methodologies for environmental monitoring and agricultural technology.

With a Master's degree in Earth System Science and Engineering from IIT Guwahati and a Bachelor's in Civil Engineering, I combine geospatial analysis with machine learning to solve real-world problems in agriculture, disaster management, and environmental monitoring.

My passion lies in democratizing access to geospatial data through innovative web applications and building scalable solutions for precision agriculture and environmental risk assessment.

Featured Projects

SpectralNi30 - GIS Analytics Engine

Python • Streamlit • Google Earth Engine API

A full-stack geospatial web application democratizing access to optical and SAR data. Features include real-time Vegetation Health Index monitoring, soil moisture retrieval, flood mapping, and automated time-series charting of crop growth cycles.

Live Demo GitHub

geoFormatX - Vector Interoperability Engine

Python • Streamlit • Geopandas • Rasterio

A serverless geospatial ETL web application automating conversion of vector datasets between Shapefile, GeoJSON, and KML formats. Solves CRS reprojection issues for seamless interoperability between legacy cadastral data and modern precision agriculture tools.

Live Demo GitHub

Machine Learning-Based Crop Yield Forecasting

Python • Sentinel-1/2 • Machine Learning

Developed predictive models for crop yield forecasting using thermal and optical remote sensing data. Built data pipelines to predict sugarcane yield across growth stages, assisting farmers with targeted irrigation and nutrient planning.

GitHub

Environmental Risk Assessment using Hyperspectral Data

Python • PRISMA • Machine Learning

Applied machine learning models for surface anomaly detection using PRISMA hyperspectral datasets. Demonstrated utility of narrow-band spectral analysis for identifying stress markers in vegetation and soil contamination.

GitHub
